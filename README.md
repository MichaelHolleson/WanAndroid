# Michael's WanAndroid

## 一、简介：

本项目使用鸿祥大神WanAndroid的开放API，采用Material Design + MVP + Retrofit + Dagger2 + RxJava + GreenDao + Glide
等开源框架和技术，仅供学习和交流。

## 二、开放API：
[WanAndroid开放API](https://www.wanandroid.com/blog/show/2)

## 三、页面设计：
### 3.1、模块设计
- 首页：banner、文章列表
- 知识体系：知识体系、导航
- 项目：项目
- 我：登录、注册、退出、收藏、分享、系统设置、积分、TODO
- 搜索：热门搜索
